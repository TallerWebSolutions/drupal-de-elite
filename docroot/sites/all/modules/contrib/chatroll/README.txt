Chatroll is an easy to use and flexible Drupal chat module. 
It can be used as a live chat or shoutbox, anywhere on your site. 

This module is for Drupal 7 only.
To download it's Drupal 6 version, go to http://www.drupal.org/project/chatroll 


Installation
------------

1. Copy the folder named 'chatroll' and its contents to the modules directory
   of your Drupal installation (for example 'sites/all/modules/').

    -- then --
2. Go to 'admin/modules' and enable chatroll.

3. Go to 'admin/config/services/chatroll' and edit settings for the chatroll
   to your need.
